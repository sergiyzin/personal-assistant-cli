# Personal Assistant CLI

Це є командний фінальний проєкт нашої команди "Team 4" для курсу Python Programming: Foundations and Best Practices.

# Функціонал

Програма вміє:

- додавати контакти
- зберігати ім'я, адресу, телефони, email і день народження
- шукати контакти
- редагувати та видаляти контакти
- показувати найближчі дні народження
- додавати нотатки
- шукати, редагувати та видаляти нотатки
- додавати теги до нотаток
- шукати та сортувати нотатки за тегами
- зберігати дані на диску

# Запуск через Terminal

python main.py
