# Personal Assistant CLI

Командний фінальний проєкт команди **Team 4** для курсу
**Python Programming: Foundations and Best Practices**.

Програма — це консольний персональний помічник для роботи з контактами та нотатками.
Дані зберігаються локально у JSON-файлі, тому після перезапуску програми інформація не втрачається.

---

# Функціонал

Програма дозволяє:

### Контакти
- додавати контакт
- редагувати ім’я контакту
- додавати, редагувати та видаляти телефони
- зберігати email
- зберігати адресу
- додавати дату народження
- шукати контакт
- видаляти контакт
- показувати список контактів
- показувати найближчі дні народження

### Нотатки
- створювати нотатку
- редагувати нотатку
- видаляти нотатку
- шукати нотатки
- додавати теги
- видаляти теги
- шукати нотатки за тегами
- сортувати нотатки за тегами

### Робота з даними
- автоматичне збереження на диск
- автоматичне завантаження при запуску програми
- обробка помилок читання JSON

---

# Технології

Проєкт реалізований з використанням:

- Python 3
- Object-Oriented Programming
- JSON для збереження даних
- CLI (Command Line Interface)
- Git та GitHub для командної розробки

---
## Dependencies
The project uses only Python standard libraries.
No external packages are required.
